App Description: This application allows users to define ringer settings based on different contexts
such as Location, Time, Detected Activity and Headphone Status. The user can configure their phone
to go on Do Not Disturb, Vibrate or Normal mode whenever one of this contexts is triggered, so that
they don't have to manually change the ringer on their phone.
